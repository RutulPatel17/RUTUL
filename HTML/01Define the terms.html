Website:- Website is a collection of web pages and related content over the internet.

Web Page :- Web page is a document file that is wrriten in HTML and can viwed by on any web browser.

Web Browser :- Web browser is a software application for accessing data or web pages throgh the internet.

Web Server:- Web server is a software and hardware that uses HTTP and other protocols to respond to client requests made over the world wide web.

HTML :- HTML stands for Hypertext Markup Language. HTML describes the structure of a web page. It contain Tags and elements. HTML elements tells the browser how to display the content.

CSS :- CSS stands for Cascading Style Sheet. CSS is the language for describing the presentation of web pages including colors, layouts and fonts.
